# Slidr

A native macOS media gallery viewer for images, GIFs, and videos.

## Features

- **Grid View**: Browse your media with adjustable thumbnail sizes
- **Video Hover Scrubbing**: Preview videos by hovering over thumbnails
- **Playlists**: Create manual playlists or smart playlists that auto-update from folders
- **Slideshow**: Fullscreen presentation with multi-monitor support
- **Keyboard-Driven**: Full keyboard control during slideshows

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac
